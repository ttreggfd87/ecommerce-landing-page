# E-commerce Landing Page

A high-conversion landing page meant for an e-commerce SaaS product. Built with **React**, **TailwindCSS**, and **Framer Motion**.

## Features
*   **Hero Section:** High-impact value proposition with animated entry.
*   **Responsive Design:** Mobile-first navbar and grid layout.
*   **Micro-interactions:** Smooth hover states and scroll-triggered animations using Framer Motion.
*   **Performance:** Optimized images and code splitting.

## Tech Stack
*   React 18
*   Vite
*   TailwindCSS
*   Framer Motion

## Setup

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Start Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build:**
    ```bash
    npm run build
    ```
